## Что такое переменная

Переменная — это именованная область памяти, в которой хранится значение определённого типа.
В C# тип переменной указывается явно при объявлении.

## Базовые типы данных

В языке C# есть несколько встроенных типов данных, которые покрывают большую часть задач:

- `int` — целое число (от -2 147 483 648 до 2 147 483 647)
- `double` — число с плавающей точкой
- `string` — строка
- `bool` — логическое значение `true`/`false`
- `char` — одиночный символ

```csharp
int age = 25;
double price = 19.99;
string name = "Alice";
bool isActive = true;
char letter = 'A';
```

## Ввод и вывод

Стандартные средства консольного ввода/вывода:

```csharp
Console.WriteLine("Введите ваше имя:");
string? name = Console.ReadLine();
Console.WriteLine($"Привет, {name}!");
```

Это базовый шаблон, с которого начинается любая консольная программа.
